FAQs

How much work is hosting?

Start small. One pickup window per week. The system handles ordering and aggregation so hosting stays manageable.

Do we need a large community?

No. Consistency matters more than size. Many groups start with a small core and grow through reliability.

Can we support multiple producers?

Yes. REKO is built for multi-producer ordering and a single coordinated pickup flow.
